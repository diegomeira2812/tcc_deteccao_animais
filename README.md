# Detecção de Macacos-Prego com Deep Learning

Este repositório contém o código e os experimentos realizados no Trabalho de Conclusão de Curso (TCC), cujo objetivo é comparar o desempenho dos modelos ResNet18 e EfficientNetB0 na tarefa de detecção de macacos-prego (Sapajus sp.) em imagens.

O estudo se insere na área de aprendizado profundo aplicado à conservação ambiental, explorando como arquiteturas de redes neurais convolucionais podem apoiar o monitoramento automatizado da fauna silvestre.

# Estrutura do  Projeto

├── .vscode/                  # Configurações do ambiente de desenvolvimento
├── deteccao/                # Códigos de detecção com ResNet18 e EfficientNetB0
├── metrics_outputs/         # Relatórios e métricas dos treinamentos
├── treinamento_modelos/     # Scripts para treinamento dos modelos
├── avaliar_resnet18.py      # Script de avaliação do modelo ResNet18
├── cuda.py                  # Verificação e configuração de GPU
├── README.md                # Este arquivo
